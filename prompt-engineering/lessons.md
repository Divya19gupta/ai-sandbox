# LESSON-1:
---------
1) Difference between Base LLM and Intructional LLM is that, the former is trained on large amount of texts, and in later we take advantage of that and train the model with both input and output using the concept of Re-inforcement Leanring to make the LLM harmless, and behave in a certain way.
2) Base LLM: It tends to predict what next word would be
3) Instructional LLM: These are the LLMs which answer to your question/query

Example: Was ist die Hauptstadt von France? BL will scrape the internet and would find more questions on france like, Wo ist France? Geschite von France? etc while the IL will give an asnwer: Paris.
